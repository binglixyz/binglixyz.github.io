Text normalization

Posted on 2017-12-05

Overview

There are 3 parts of text normalization, listed as follows:

  • Tokenization
    • Usage: Text -> Words
    • Example: “I love you” -> “I”, “love”, “you”
  • Lemmatization (includes stemming)
    • Usage: Words -> Lemma (stemming is just strip suffixes)
    • Example: “sings”,”sang”,”sung” -> “sing”
  • Sentence segmantation
    • Usage: Text -> Sentence
    • Example: “You are such a lovely girl. Come on. Dress on.” -> “You are such a lovely girl.”, “Come on.”, “Dress on.”

Words and Corpora

  • Corpus: a computer readable collection of text
  • Utterence
    • Disfluency
      • Fragment: Words like “main-mainly”
      • Filled pause: “uh”, “em”
    • Capitalize
  • Wordform: “sings”, “sang”
  • Measure of number of words
    • Word type: Types is the number of distinct words, denoted as $|V|$
    • Word token: Tokens is the total number of running words, denoted as $N$
    • Number of lemmas
